# Manual Testing Test Cases

This repository contains sample manual testing artifacts created for web and mobile applications, covering core QA activities such as functional testing, regression testing, validation testing, and defect reporting.

## Repository Overview

The purpose of this repository is to demonstrate practical manual testing knowledge, structured test case design, and QA documentation practices used in real-world software testing projects.

## Modules Covered

- Login
- Registration
- Checkout & Payment
- User Profile
- Search & Filter

## Test Coverage

The test cases included in this repository cover:

- Functional Testing
- Regression Testing
- UI/UX Validation
- Negative Testing
- Boundary Value Testing
- Validation Testing
- Security Testing
- Mobile Responsiveness Testing
